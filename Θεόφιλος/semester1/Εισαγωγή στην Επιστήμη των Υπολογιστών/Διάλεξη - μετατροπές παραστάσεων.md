### Προηγούμενη διάλεξη: 
Στην προηγούμενη διάλεξη έδειξε για το πως έυκολα μπορούμε με αριθμητικούς μεθόδους να βρούμε τον τον δεκαδικό αιρθμό για το νούμερο $375_{10}$:

Αριθμ. Πραξ.     Αποτελ.       Υπολ
375 / 2           =   187               1
187 / 2           =   93                 1
93  / 2            =   46                 1
46 / 2             =   23                 0
23 / 2             =   11                 1
11 / 2             =   5                   1
5 / 2               =   2                   1
2 / 2               =   1                   0
1 / 2               =   0                   1

Εφόσον κάναμε τις διαδοχικές διαρέσεις και βρήκαμε τα υπόλοιπα. Παίρνουμε τα υπόλοιπα και τα διαβάζουμε ανάποδα, οπότε ο δυαδικός αντίστοιχος του αριθμού $375_{10}$ είναι $101110111_{2}$ 

Με τον παραπάνω αλγόριθμο μπορούμε να βρούμε και την αναπαράσταση αριθμών και άλλων συστημάτων, όπως το εξαδικό, οκταδικό δεκαεξαδικό κ.α.

Αρκεί όταν διαιρούμε να διαιρούμε με την βάση του συστήματος μας. (έστω $375_{10}$ και θέλουμε σε οκταδικό τότε 375 / **8**)

Παράδειγμα:
139 / 16 = 8      Υπολ: 11
8 / 16 = 0          Υπολ: 8


Για να βρούμε το δεκαδικό μέρος ενός αριθμού, αρκεί να πολλαπλασιάζουμε αυτό το δεκαδικό μέρος μέχρι να βρούμε $1.0$. Στην περίπτωση που δεν μπορούμε να βρούμε 1.0 τότε σημαίνει πως γίνεται κάποια επανάληψη ψηφίων, άρα το σταματάμε πιο νωρίς και λέμε πως ο δεκαδικό αριθμός που βρήκαμε είναι προσέγγιση. Το υπόλοιπο σε αυτήν την περίπτωση είναι ότι είναι δεξιά από την υποδιαστολή.

Παράδειγμα:

Αριθμ. Πραξ.     Αποτελ.       Υπολ
.65625 * 2   =   1.31250       1
.31250 * 2   =   0.6250         0
.6250 * 2     =   1.25              1
.25 * 2         =   0.5                0
.5 * 2           =   1.0                1

Σε αυτήν την περίπτωση τώρα που βρήκαμε τα υπόλοιπα, τα παίρνουμε απο πάνω πρως τα κάτω. άρα ο δεκαδικός αριθμός $0.65625_{10}$ μπορεί να αναπαρασταθεί στο δυαδικό ως $.10101_{2}$

Παράδειγμα προσέγγισης:

Αριθμ. Πραξ.     Αποτελ.       Υπολ
.1 * 2           =   0.2                0
.2 * 2           =   0.4                0
.4 * 2           =   0.8                0
.8 * 2           =   1.6                1
.6 * 2           =   1.2                1
.2 * 2           =   0.4                0
.4 * 2           =   0.8                0
.8 * 2           =   1.6                1
.6 * 2           =   1.2                1
.2 * 2           =   0.4                0

Σε αυτήν την περίπτωση μπορούμε να δούμε πως υπάρχει ένα μοτίβο, άρα με προσέγγιση μπορούμε να πούμε πως ο δεκαδικός αριθμός $0.1_{10}$ έχει προσέγγιση δυαδική ως εξής $.0001100110\dots_{2}$

Άμα θέλουμε να μετατρέψουμε έναν δεκαδικό αριθμό σε δυαδικό, δηλαδή έναν αριθμό που έχει και ακέραιο μέρος και δεκαδικό. Τότε το χωρίζουμε στα δύο αντίστοιχα μέρη που ανάφερα (ακέραιο και δεκαδικό) και εφαρμόζουμε την θεωρία που έχει προαναφερθεί.

### Αυτή η διάλεξη - Μη αριθμιτικές μέθοδοι μετατροπής

Μιλάμε για μετατροπές δυαδικού -> οκταδικού, δεκαεξαδικού
Γιατί   $2^3=8,\ 2^4=16$

Κάθε οκταδικό ψηφίο αντιστοιχή σε 3 δυαδικά ψηφία.

Οπότε για παράδειγμα, άμα θέλουμε να κάνουμε την παρακάτω μετατροπή:

$101110110_{2}=(?)_{8}$

101 = 5
110 = 6
110 = 6

Άρα $566_{8}$ ειναι η σωστή οκταδική αναπαράσταση του δυαδικού αριθμού.

Ομοίως θα εφαρμόσουμε την ίδια διαδικασία στα αντίστοιχα μέρη ενώς αριθμού που έχει ακέραιο και δεκαδικό μέρος.

> Όταν δεν μπορούμε να φτιάξουμε αρκετά ζευγαράκια, τότε προσθέτουμε "άχρηστα" μπιτς. Δηλαδή πρόσθεσε μηδενικά για το ακέραιο μέρος απο τα αριστερά, και αντίστοιχα απο τα δεξιά για το δεκαδικό μέρος.

Την ίδια λογική χρησιμοποιούμε και ανάποδα. Για κάθε ψηφίο του οκταδικού αριθμού που μας δίνεται ορίζουμε 3 ψηφία δυαδικού.

άρα για τον παρακάτω οκταδικό αριθμό:

$732.2005_{8}=(?)_{2}$

7 = 111
3 = 011
2 = 010
.
2 = 010
0 = 000
0 = 000
5 = 101


άρα άμα τα βάλουμε όλα τα νούμερα στην σειρά θα πάρουμε: 

$111\ 011\ 010.010\ 000\ 000\ 101_{2} = 732.2005_{8}$

Για μετατροπή δυαδικού στο δεκαεξαδικό αρκεί απλά, αντί για να τα ομαδοποιήσουμε σε τριάδες, να τα ομαδοποιήσουμε σε τετράδες.

Άμα θέλουμε να μετατρέψουμε απο δεκαεξαδικό σε οκταδικό και αντίστροφα αρκεί απλά **ενδιάμεσα** να μετατρέψουμε τους αριθμούς στο δυαδικό.

#### Πορσθαφαίρεση μη προσημασμένων ακεραίων

Δυαδική πρόσθεση είναι όπως φαίνεται παρακάτω

```
+| 0 | 1 |
0| 0 | 1 |
1| 1 |10 |
```

Παρατηρούμαι παραπάνω πως όταν προσθέτουμε $1_{2}+1_{2}=10_{2}$ 

Ουσιαστικά μπορούμαι να ονομάσουμε αυτό το εξτρά μπιτ που εμφανήστηκε ($1$) ως κρατούμενο (το ίδιο μπορούμε να παρατηρήσουμε και στην κανονική πρόσθεση, όπως π.χ. $9+5=14$)

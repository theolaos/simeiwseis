## Μέση ταχύτητα και Επιτάχυνση:

$$\overline{\vec{u}} = \frac{\Delta \vec{r}}{\Delta t},\ \ \ \ \  \overline{\vec{a}} = \frac{\Delta \vec{u}}{\Delta t}$$
## Στιγμιαία ταχύτητα και επιτάχυνση:

Είναι η παράγωγος ως προς τον χρόνο της διανυσματική θέσης και ταχύτητας, άρα:
$$\vec{u} = \frac{d\vec{r}}{dt}, \ \ \ \ \ \vec{a} = \frac{d\vec{u}}{dt}$$

# Κίνηση βλήματος
## Στην ελεύθερη πτώση:

Ορίζουμε ότι $a=g=9.8\frac{m}{s^2}$, η οριζόνται και η κάθετη κίνηση θεωρούνται ανεξάρτητες.

Με την προηγούμενη θεώρηση, λέμε επίσης: 
$$u_{x}=u_{x0}$$
$$u_{y} = u_{y_{0}} - gt$$
Και ισχύουν οι παρακάτω τύποι.

$$x = x_{0} + u_{x0}t$$
$$y = y_{0} + u_{y0}t - \frac{1}{2}gt^2$$
## Κίνηση του βλήματος:

Η κίνηση του βλήματος είναι μια παραβολή.

Οπότε από τα μαθηματικά του λυκείου μπορούμε να συμπαράνουμε πως, το μέγιστο σημείο του βλήμματος θα είναι στο χρονικό σημείο:

$$t_{μ} = \frac{u_{y0}}{gt^2}$$
Και έπειτα το μέγιστο σημείο είναι $(x(t_{μ}),\ \ y(t_{μ}))$

Σε περίπτωση που δουλεύουμε με διανύσματα, τότε $u_{y0} = |\vec{u}|\cdot \sin \phi$, όπου $\phi$ είναι η γωνία του διανύσματος. Ή μπορούμε να αναλύσουμε το διάνυσμα και τότε άμα αναλύσουμε το $\vec{u} = x\hat{i}+y\hat{j}$ τότε $u_{y0} = x \hat{i}$.

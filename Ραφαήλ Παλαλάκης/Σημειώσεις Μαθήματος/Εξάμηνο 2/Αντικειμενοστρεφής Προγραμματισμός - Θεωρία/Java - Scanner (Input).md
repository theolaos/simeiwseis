### **Use 'Scanner' inside a java project**

We import the class 'Scanner' with :

```java
immport java.util.Scanner;
```


And declare the new object scanner with the input :

```java
Scanner inputLine = new Scanner(System.in);
```


---

### **Scanner Methods**

```java
nextLine() // Seeks input values untill it finds character '\n'
next() // Seeks input values until the next String
nextInt() // Seeks input of the next int
nextDouble() // Seeks input of the next Double
```

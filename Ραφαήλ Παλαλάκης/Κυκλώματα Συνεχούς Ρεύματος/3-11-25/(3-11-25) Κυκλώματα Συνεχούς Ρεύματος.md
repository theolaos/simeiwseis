##### **Διαιρέτης Τάσης**

$V_1 = \frac{R_1}{R_1+R_2} * V$
$V_2 = \frac{R_2}{R_1 + R_2}*V$
$V_n = \frac{R_n}{R_1 + R_2 + ... + R_N} * V$

![[Κυκλώματα Συνεχούς Ρεύματος/3-11-25/Images/ΠΑΡ1.png]]

Ο διαιρέτης τάσης χρησιμοποιείται σε κόμβους που είναι σε σειρά

##### **Διαιρέτης Ρεύματος**

$I_1 = \frac{R_2}{R_1 + R_2} * i$
$I_2 = \frac{R_1}{R_1 + R_2} * i$
